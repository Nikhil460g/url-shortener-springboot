# URL Shortener with Click Analytics

## Overview

A URL Shortener Web Application built using Spring Boot, MySQL, Hibernate, and Thymeleaf.

Users can:

* Generate short URLs
* Create custom short URLs
* Redirect to original URLs
* Track click counts
* View URL statistics
* View URL history
* Delete URLs

## Technologies Used

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA
* Hibernate
* MySQL
* Thymeleaf
* HTML
* CSS

## Features

* URL Shortening
* Custom Short Codes
* Click Tracking
* Statistics Dashboard
* URL History
* Delete URL Functionality

## Database

MySQL Database used for storing URL information and analytics.
